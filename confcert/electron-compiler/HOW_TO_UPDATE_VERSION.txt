# How to Update Version

## ⚡ Quick Update (Only 1 File!)

To update the app version, edit **ONLY** this file:

**`version.js`** - Line 4

```javascript
const APP_VERSION = "1.1.3";  // ← Change this
```

That's it! All other files will automatically use this version.

---

## 🔄 What Happens Automatically

When you change `version.js` and run the app:

1. ✅ `package.json` version is auto-synced
2. ✅ Header badge shows new version
3. ✅ Terminal log shows new version
4. ✅ Tooltip shows new version
5. ✅ Installer filename uses new version

---

## 🚀 Commands

All commands automatically sync the version:

```bash
npm start              # Syncs version, then starts app
npm run dev            # Syncs version, then starts in dev mode
npm run build:win      # Syncs version, then builds installer
```

---

## 📦 Files Affected

- `version.js` ← **EDIT THIS ONLY**
- `package.json` ← Auto-updated
- `index.html` ← Reads from version.js
- `renderer.js` ← Reads from version.js

---

## ✅ Example

**Before:**
```javascript
const APP_VERSION = "1.1.2";
```

**After:**
```javascript
const APP_VERSION = "1.1.3";
```

**Result:**
- Package version: `1.1.3`
- Header shows: `v1.1.3`
- Terminal shows: `Version: 1.1.3`
- Installer: `Solidity Playground-Installer-1.1.3.exe`

Done! 🎉
